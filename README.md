# Welcome to the to the official UI of SWAPI:ROTJ 

## Changes from previous version ui
  - More SW themed ui/ux.
  - Search results are in a styled list instead of plain json.
  - All results have images and descriptions.
  - The results themselves, as well as search functionality are now human friendly! ie: you search and read by names instead of api links or primary keys/
  - This UI is written in typescript, as opposed to Javascript.
  - Utilizes docker.

## How to Run Locally
  - If using docker, build and run the image as you would any other.
  - Otherwise, simply install the node modules (npm i) and start the application (npm start) .
